## 22.09.05
자코비안까지는 계산되지만, 가우스 뉴턴법을 수행하려 하면 2번째 반복에서 역행렬 계산 중 nan이 나오는 문제 발생.  
```
[[1, 7.4, 0, 0.00345946],
 [39.0625, 289.063, 1, 0]]
```
가우스 소거법을 진행하다 보면 2행 2열이 0이 되는 문제가 발생한다(7.4 * 39.0625를 곱하면 정확히 289.063이 나온다). 이런 경우 어떻게 예외를 설정해야할까..?
